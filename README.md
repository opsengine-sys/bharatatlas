
# 🇮🇳 Bharat Atlas

Bharat Atlas is a small experimental project where I explored **building websites using AI tools**, specifically using **Lovable**.

The idea was to see how quickly I could go from concept → working product by leveraging AI-assisted development. I chose a travel/exploration theme to build something practical while learning the workflow.

This app helps users discover places across India—tourist spots, restaurants, hotels, and more—while also serving as a hands-on experiment in modern AI-driven development.

---

## 🚀 What This Project Is About

* Exploring **AI-powered website building**
* Learning how tools like **Lovable** can speed up development
* Rapid prototyping with minimal manual coding
* Understanding how AI can structure UI, flows, and features

---

## ✨ Features

📍 **Discover Nearby Places**
Find tourist attractions, restaurants, hotels, and important locations around you

🗺️ **Explore India**
Browse famous landmarks and hidden gems across the country

🧭 **Smart Categorization**
Locations organized into simple, intuitive categories

📊 **Basic Place Insights**
Quick information to help with decisions

⚡ **Simple & Clean Navigation**
Built with focus on usability and speed
